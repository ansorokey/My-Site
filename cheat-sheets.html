Math Class

JavaScript methods and properties: Math Class
Math can be referred to statically, does not need to have an instance created
Math.PI -A constant value, approx. 3.14159
Math.abs(x) -returns the absolute value of x
Math.ceil(x) -returns the nearest integer (whole number) rounded up
Math.floor(x) -returns the nearest integer (whole number) rounded down
Math.max(x) -returns the largest value of a series of numbers -requires at least two comma separated arguments
Math.min(x) -returns the lowest value in a series of numbers -requires at least two comma separated arguments
Math.pow(x, y) -returns the value of X raised to the Y power
Math.random() -returns a decimal number FROM (including) 0 and UP TO (excluding) 1 -multiply and then round your results using Math.floor() or Math.ceil() to return an integer
Math.round() -returns the nearest integer value
Math.sign() -returns whether the value is positive, negative, or zero

string methods and properties

The JavaScript string class has both properties and methods
string.length -is a property, returns the total number of characters inside a string
string.charAt() -returns the character at the index specified
string.concat() -returns a new string that is the combination of base string + 1 or more string arguments comma separated
string.includes() -returns a boolean value if the base string contains the value of the argument string
string.endsWith() -returns a boolean value if the string ends with the specified argument
string.indexOf() -returns the starting index position of the string passed in -returns -1 if not found
string.replaceAll(replace, with) -searches for and replaces all values x with second value Y
string.slice() -returns a string from (inclusive) one index to (exclusive) and ending index
string.split() -returns an array of string separated at the passed argument
string.startsWith() -returns a bopolean if the string begins with the specified argument
string.substring() -returns a string from starting index (inclusive) to ending index (exclusive)
string.toLowerCase() -returns the string in lowercase
string.toUpperCase() -returns the string in uppercase
string.trim() -returns string with whitespace at the front end and of string removed
string.value)f() -returns the primitive value of the character at index
