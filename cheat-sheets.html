Math Class

JavaScript methods and properties: Math Class Math can be referred to statically, does not need to have an instance created Math.PI -A constant value, approx. 3.14159 Math.abs(x) -returns the absolute value of x Math.ciel(x) -returns the nearest integer (whole number) rounded up Math.floor(x) -returns the nearest integer (whole number) rounded down Math.max(x) -returns the largest value of a series of numbers -requires at least two comma separated arguments Math.min(x) -returns the lowest value in a series of numbers -requires at least two comma separated arguments Math.pow(x, y) -returns the value of X raised to the Y power Math.random() -returns a decimal number FROM (including) 0 and UP TO (excluding) 1 -multiply and then round your results using Math.floor() or Math.ceil() to return an integer Math.round() -returns the nearest integer value Math.sign() -returns whether the value is positive, negative, or zero
